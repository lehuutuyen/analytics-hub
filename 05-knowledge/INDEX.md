# Knowledge — Data Analytics

Kiến thức nền tảng về phân tích dữ liệu, phục vụ cho việc ra quyết định chiến lược.

## Cấu trúc

Flat files đánh số + INDEX tổng quan. Mỗi file = 1 module kiến thức hoàn chỉnh.

```
05-knowledge/
├── INDEX.md          ← Bản đồ + outline tất cả module
├── 01-xxx.md         ← Module 1
├── 02-xxx.md         ← Module 2
└── ...               ← Thêm file khi học thêm
```

## Template mỗi module

```markdown
## 1. Bản chất (Khái niệm lõi)
## 2. Lý do tồn tại (Đang giải quyết vấn đề gì?)
## 3. Chi tiết chiến lược
## 4. Áp dụng vào dự án (lehuutuyen.com / Pawvibe / YouTube)
## Nguồn
```
