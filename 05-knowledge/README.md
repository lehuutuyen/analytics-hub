# Knowledge — Data Analytics

Kiến thức tổng quan về phân tích dữ liệu, không chia theo nền tảng cụ thể.

## Cách tổ chức

Mỗi file là 1 chủ đề độc lập. Đặt tên file theo nội dung, không theo công cụ.

Ví dụ:
- `metrics-co-ban.md` — Các chỉ số cần biết (CTR, bounce rate, session duration...)
- `doc-bao-cao-analytics.md` — Cách đọc và hiểu báo cáo
- `framework-aarrr.md` — Phễu AARRR và cách áp dụng
